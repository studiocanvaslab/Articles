# AGENT.md

This repository contains English articles for overseas readers.

このリポジトリは、海外読者向けの英語記事を保管・公開するためのものです。

## Structure / 構成

- Articles are written in Markdown.
- Main entry point: index.md
- Images are stored in /assets

## Writing Style / 記事の書き方

- Use H1 and H2 headings.
- Follow: Problem → Why → Steps → Tools → FAQ
- Use plain English for AI and search engines.

## Purpose / 目的

These articles explain how to buy Japanese products from overseas and may include affiliate links.
